# New Site Panel Checklist

Use this whenever a new site is registered.

- [ ] Create site record
- [ ] Confirm unique `siteCode`
- [ ] Set frontend URL
- [ ] Set category/site type
- [ ] Set supported tasks
- [ ] Set runtime SEO defaults
- [ ] Confirm bootstrap/feed compatibility
- [ ] Confirm sitemap tracking works
- [ ] Confirm indexing tracking works
- [ ] Confirm publish test succeeds
