# Site Creation Workflow

## Objective
Use Master Site Panel as the operations system for every new site created from the automation template.

## Workflow
1. Create new site repo from template.
2. Configure identity and branding in the site.
3. Deploy the site.
4. Register the site in Master Site Panel.
5. Verify feed/bootstrap connectivity.
6. Verify sitemap and indexing tracking.
7. Verify SEO overrides and audit visibility.

## Registration minimums
- unique `siteCode`
- frontend URL
- category/site type
- supported tasks
- runtime SEO defaults

## After registration
- publish one test item
- check sitemap visibility
- check indexing status
- check search and detail page rendering
